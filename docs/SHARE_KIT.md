# Project Share Kit

This file gives you ready-to-use text and a simple checklist for publishing the project on LinkedIn and GitHub.

## Best Post Assets

Use 3 to 5 visuals in this order:

1. Clean homepage screenshot of the Streamlit app
2. Rainfall chance curve section from the app
3. Top rainy days ranking table or panel
4. Forecast chart from `outputs/figures/future_forecast_march_october_2026.png`
5. Optional repository screenshot showing the clean folder structure

## Best Caption Angle

Lead with this idea:

`This project does not stop at monthly rainfall forecasting. It also estimates which days in a selected month are most likely to receive rainfall.`

That line makes the project feel more product-oriented and more memorable.

## GitHub Repo Description

Use this as the GitHub repository description:

`Leakage-free rainfall forecasting pipeline and Streamlit app for monthly forecasts and day-level rainfall chance ranking.`
